## 第一章
### 打胶为尊の世界
引子：世界是什么？世界的真理是什么？屎肥狗在出年生就在思考，直到他摸到了他的发射器时才明，世界就是打胶！！！
<!-- .slide vertical=true -->
### ●“胶之力，三段！”在机器发出报结后，肥狗在一阵嘲笑中拿下了自己的评分，脸色瞬间变得铁青，“本不应该是这样的！”肥狗心中暗骂道，眼中闪烁着不甘的光芒，“没眼的老天！” 
<!-- .slide vertical=true -->
### ●谁能想到，就在一个月前，肥狗还是个胶之力大成者，一身肌肉虬结，本是一代胶术之子，享受着无数人的敬仰和羡慕。但为何如今胶之力后退，沦为了个废物，身材也逐渐变得臃肿不堪？就在他自暴自弃，整日沉迷于低俗真人片时，下体中的肛塞突然振动起来。不可能！他的胶之力已经无法使用了，为何会这样？
<!-- .slide vertical=true -->
### ●此时，一个身形佝偻但眼神锐利的老人突然出现在肥狗身旁，他竟是肥狗的爷爷——屎玖亿，传说中达到了胶尊之境的男人，被称为打胶终焉的传奇人物。他的脸上布满了岁月的痕迹，但眼中却闪烁着智慧的光芒。
<!-- .slide vertical=true -->
### ●“肥狗！”爷爷的声音低沉而有力，“你阳痿不举是遇到了瓶颈，看那些低俗真人片的你是无法突破的。只有观看传说中的二次元核片，方可达到突破，一举成为胶师之境。”说着，他从怀中掏出两样东西，正是两种中器（飞机杯和润滑油），“爷爷再赐你这两样东西，危机之际可帮你。你只需为我报仇，打败一个叫茶叶蛋的人便可。”
<!-- .slide vertical=true -->
### ●肥狗闻言，眼中闪过一丝坚定。他接过爷爷手中的东西，深深地鞠了一躬，“爷爷，我一定会为你报仇的！”至此，肥狗便踏上了前往东坤学院的路途，去寻找那传说中的二次元核片和那个叫茶叶蛋的男人。
